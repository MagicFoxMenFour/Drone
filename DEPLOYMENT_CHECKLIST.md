# Чеклист развертывания на Amvera

Используйте этот чеклист при развертывании вашего сайта на Amvera.

## ✅ До развертывания

### Подготовка репозитория

- [ ] Код залит в GitHub
- [ ] Используется основная ветка (main/master)
- [ ] Создана ветка `content` в GitHub
- [ ] `.env` файл добавлен в `.gitignore`
- [ ] `package.json` имеет скрипты build и dev

### Учетные данные

- [ ] GitHub PAT токен создан (Personal Access Token)
- [ ] Токен имеет права на `repo` или `contents`
- [ ] Токен не истек и рабочий
- [ ] Resend API ключ подготовлен (если используется email)
- [ ] Домен верифицирован на Resend (если используется)

### Админ панель

- [ ] Выбран сильный пароль (минимум 12 символов)
- [ ] Логин выбран (по умолчанию `admin`)
- [ ] Сгенерирована строка ADMIN_SESSION_SECRET

Генерирование:
```bash
node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"
```

## 🚀 При развертывании на Amvera

### Создание проекта

- [ ] Зарегистрирован аккаунт на Amvera (https://console.amvera.io)
- [ ] Создан новый проект
- [ ] Выбран GitHub как источник
- [ ] Авторизирован доступ к GitHub репозиторию
- [ ] Выбрана ветка `main` или `master`

### Конфигурация

- [ ] Build command: `npm install && npm run build`
- [ ] Artifact path: `dist`
- [ ] Node.js версия: 18+ LTS

### Переменные окружения

Добавьте в Amvera Console → Environment Variables:

- [ ] `ADMIN_LOGIN` = admin (или выбранный логин)
- [ ] `ADMIN_PASSWORD` = ваш_пароль (сильный!)
- [ ] `ADMIN_SESSION_SECRET` = сгенерированная строка
- [ ] `GITHUB_OWNER` = ваш_username_github
- [ ] `GITHUB_REPO` = название_репо
- [ ] `GITHUB_TOKEN` = ваш_PAT_токен
- [ ] `GITHUB_CONTENT_BRANCH` = content
- [ ] `RESEND_API_KEY` = (опционально)
- [ ] `RESEND_FROM` = (опционально)
- [ ] `LEADS_TO_EMAIL` = (опционально)

### Развертывание

- [ ] Нажата кнопка Deploy
- [ ] Следили за логами сборки
- [ ] Сборка завершилась без ошибок
- [ ] Сайт доступен по ссылке

## 🚨 Решение проблем

Если что-то не работает:

### Сборка не проходит

1. Проверьте логи в Amvera Console → Logs
2. Запустите локально: `npm run build`
3. Убедитесь что Node.js версия 18+

### Админ панель не доступна

1. Проверьте переменные окружения в Amvera
2. Убедитесь что API запросы идут к `/api/admin/*`
3. Откройте DevTools (F12) → Network → смотрите ошибки

### GitHub интеграция не работает

1. Проверьте `GITHUB_TOKEN` в переменных
2. Убедитесь что токен не истек
3. Создайте новый токен с правами `repo`

### Email не отправляются

1. Проверьте `RESEND_API_KEY`
2. Убедитесь что домен верифицирован на Resend
3. Проверьте `LEADS_TO_EMAIL`
