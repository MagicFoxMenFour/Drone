## 🚀 Как начать

### Вариант 1: Быстрый старт (локально)
```bash
npm install
cp .env.example .env
# Отредактируйте .env

npm install -g vercel
vercel dev

# Откройте http://localhost:3000/admin/login
```

### Вариант 2: Развертывание на Amvera (рекомендуется)
1. Откройте `AMVERA_DEPLOYMENT.md`
2. Следуйте шагам
3. Займет ~5 минут!

## 📍 После развертывания

Ваша админ панель будет доступна по адресу:
```
https://yourdomain.amvera.io/admin/login
```

Структура админ панели:
```
/admin              → Дашборд
/admin/services     → Управление услугами
/admin/cases        → Управление кейсами
/admin/blog         → Управление блогом
/admin/about        → Управление "О нас"
/admin/employees    → Управление сотрудниками
/admin/leads        → Просмотр заявок
```

## ❓ Частые вопросы

**Q: Где логин/пароль?**
A: В переменных окружения (`.env` локально, Environment Variables на Amvera)

**Q: Как добавить сотрудника?**
A: `/admin/employees` → нажать "Добавить" → заполнить форму

**Q: Как получить заявки по почте?**
A: Настроить Resend (смотри AMVERA_DEPLOYMENT.md)

**Q: Как восстановить данные?**
A: Всё в GitHub - просто откатите коммит

**Q: Сколько стоит хостинг?**
A: Amvera имеет бесплатный tier, GitHub бесплатен, Resend имеет free план
